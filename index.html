⚛️ Valències dels Elements

📖 Què és la VALÈNCIA?

La valència és el nombre d'electrons que un element pot perdre, guanyar o compartir per formar un compost químic.

n = mínim de liens covalents o càrrega iònica
La valència determina com els elements es combinen!
  • Metalls: Perden electrons → valències POSITIVES (+1, +2, +3...)
  • No-metalls: Guanyen electrons → valències NEGATIVES (-1, -2, -3...)
  • Transitorials: Pueden tener MÚLTIPLES valencies

⚡ Com es determina la VALÈNCIA?

La posició a la Taula Periòdica determina la valencia:

  • Grup 1 (Alcalins): +1 (1 electron a la darrera capa)
  • Grup 2 (Alcalinoterris): +2 (2 electrons a la darrera capa)
  • Grup 13 (Boro, Alumini): +3 (3 electrons a la darrera capa)
  • Grup 14 (Carboni, Silici): +4 o -4 (4 electrons a la darrera capa)
  • Grup 15 (Nitrogen, Fòsfor): -3 o +3, +5 (5 electrons)
  • Grup 16 (Oxigen, Sofre): -2 o +4, +6 (6 electrons)
  • Grup 17 (Halògens): -1 o +1, +3, +5, +7 (7 electrons)
  • Grup 18 (Gasos nobles): 0 (no es combinen normalment)

🎯 Com USAR les valencies per fer fórmules

Quan combinem dos elements, les seves valències han de "balancejar-se":

REGLA: El producte de la valencia × nombre d'àtoms ha de ser igual per als dos elements.

Exemples:

  • Na (+1) + Cl (-1): 1×1 = 1×1 → NaCl ✓
  • Ca (+2) + Cl (-1): 1×2 = 2×1 → CaCl₂ ✓
  • Al (+3) + Cl (-1): 1×3 = 3×1 → AlCl₃ ✓
  • Fe (+3) + O (-2): 2×3 = 3×2 → Fe₂O₃ ✓
  • Ca (+2) + (OH)⁻ (-1): 1×2 = 2×1 → Ca(OH)₂ ✓

💡 Trucs PER RECORDAR VALENCIES

  • GRUP = VALENCIA: Grup 1→ +1, Grup 2→ +2, Grup 13→ +3, etc.
  • Oxigen SEMPRE -2: (excepte en H₂O₂ que és -1)
  • Fluor SEMPRE -1: No canvia mai!
  • Metalls = positius: Els metalls sempre perden electrons
  • No-metalls = negatius: Els no-metalls sempre guanyen electrons
  • H amb no-metalls = +1: (HCl, H₂S, NH₃)
  • H amb metalls = -1: (LiH, MgH₂, AlH₃)
  • Transició = vàries: Fe(II) i Fe(III), Cu(I) i Cu(II)...
  • Taula Periòdica et diu tot: Mira la columna = valencia!

📚 Referència d'Hidrurs

🔥 Òxids (Compostos amb Oxigen)

📖 Què és un òxid?

Un òxid és un compost químic format per un element + oxigen. Els òxids són molt comuns a la natura.

Element + O₂ → Òxid (E₂Oₙ o EOₘ)
Depèn de les valències de l'element i l'oxigen (-2)

Hi ha dos tipus principals:

  • Òxids Metàl·lics (Òxids Bàsics): Metall + Oxigen → sòlids que formen bases
  • Òxids No-metàl·lics (Anhídrids): No-metall + Oxigen → gasos o sòlids que formen àcids

⚗️ Com es formen els òxids?

Els òxids es formen de diverses maneres:

Element + O₂ → Òxid
Combustió o combinació directa
  • Combustió simple: 2Na + O₂ → Na₂O (sodi es crema)
  • Combustió incompleta: 2C + O₂ → 2CO (monòxid de carboni)
  • Combustió completa: C + O₂ → CO₂ (diòxid de carboni)
  • Reacció d'òxids metàl·lics + aigua: Na₂O + H₂O → 2NaOH (forma base)
  • Reacció d'òxids no-metàl·lics + aigua: SO₃ + H₂O → H₂SO₄ (forma àcid)

📝 Nomenclatura dels Òxids

La nomenclatura dels òxids depèn del tipus:

🔹 Òxids Metàl·lics (Bàsics)

Sistemàtica: (prefix) + òxid + de + nom element

  • Li₂O → Monòxid de dilíti (Sist) / Òxid de liti (Stock)
  • CaO → Monòxid de calci (Sist) / Òxid de calci (Stock)
  • Al₂O₃ → Triòxid de dialumini (Sist) / Òxid d'alumini (Stock)
  • Fe₂O₃ → Triòxid de diferro (Sist) / Òxid de ferro (III) (Stock)

🔹 Òxids No-metàl·lics (Anhídrids)

Ambdós noms: Sistemàtica = Stock

  • CO₂ → Diòxid de carboni
  • SiO₂ → Diòxid de silici
  • SO₂ → Diòxid de sofre
  • N₂O₅ → Pentòxid de dinitrogen
  • Cl₂O₇ → Heptòxid de diclor (màxim clor)

🧫 Hidroxids (Bases)

📖 Què és un hidròxid?

Un hidròxid (o base) és un compost format per un metall + el grup hidroxil (OH⁻).

M(OH)n
M = metall · n = valència del metall · OH⁻ = grup hidroxil

El grup OH sempre actua amb càrrega −1. Per tant, el subíndex del grup OH coincideix amb la valència del metall.

⚗️ Com es formen?

Els hidròxids es formen quan un òxid metàl·lic reacciona amb aigua:

Òxid metàl·lic + H₂O → Hidròxid
Exemple: Na₂O + H₂O → 2 NaOH
  • El sodi (Na) té valència +1 → NaOH (1 sol OH)
  • El calci (Ca) té valència +2 → Ca(OH)₂ (2 grups OH)
  • El alumini (Al) té valència +3 → Al(OH)₃ (3 grups OH)

📝 Nomenclatura

Hi ha dues maneres d'anomenar els hidròxids:

1. Sistemàtica (IUPAC): Prefix numèric + "hidròxid de" + metall

  • NaOH → Hidròxid de sodi (mono- s'omet)
  • Ca(OH)₂ → Dihidròxid de calci
  • Fe(OH)₃ → Trihidròxid de ferro

2. Stock: "Hidròxid de" + metall + (valència en romans)

  • NaOH → Hidròxid de sodi (sense romans si només té 1 valència)
  • Fe(OH)₂ → Hidròxid de ferro (II)
  • Fe(OH)₃ → Hidròxid de ferro (III)

⚗️ Oxoàcids (Àcids Oxigenats)

📖 Què és un oxoàcid?

Un oxoàcid (o àcid oxigenat) és un compost format per hidrogen (H) + no-metall + oxigen (O).

HxXOy
H = hidrogen · X = no-metall (N, S, Cl, P...) · O = oxigen

La nomenclatura dels oxoàcids depèn de la valència del no-metall. Utilitzem sufixos (-ós, -ic) i prefixos (hipo-, per-).

⚗️ Com es formen?

Els oxoàcids es formen quan un òxid no-metàl·lic reacciona amb aigua:

Òxid no-metàl·lic + H₂O → Oxoàcid
Exemple: Cl₂O₇ + H₂O → 2 HClO₄ (àcid perclòric)
  • CO₂ + H₂O → H₂CO₃ (àcid carbònic) — Present a les begudes gasoses!
  • SO₃ + H₂O → H₂SO₄ (àcid sulfúric) — El més important industrial!
  • N₂O₅ + H₂O → HNO₃ (àcid nítric) — Ús en síntesi química.
  • P₂O₅ + H₂O → H₃PO₄ (àcid fosfòric) — Ingredient en refrescos!

📝 Nomenclatura dels Oxoàcids

La nomenclatura depèn de la valencia del no-metall central. Per als no-metalls amb multiple valencies, utilitzem prefixos i sufixos:

  • Hipo- + nom + -ós: València més baixa (exemplo: HClO, hipoclorós)
  • Nom + -ós: València intermèdia (exemple: HClO₂, clorós)
  • Nom + -ic: València més alta (exemple: HClO₃, clòric)
  • Per- + nom + -ic: València màxima (exemple: HClO₄, perclòric)

Observació: Per a no-metalls que només tenemos 2 formes (com S), no fem servir hipo- ni per-.

🧪 Hidrurs (Combinacions amb Hidrogen)

📖 Què és un hidrur?

Un hidrur és un compost químic format per hidrogen (H) combinat amb un altre element (metal o no-metal).

XHn
X = qualsevol element · H = hidrogen · n = nombre d'hidrógens

Els hidrurs es classifiquen en tres tipus segons el caràcter de l'element X:

  • Hidrurs metàl·lics: Metal + H → LiH, NaH, CaH₂
  • Hidràcids: No-metall + H → HCl, HBr, H₂S (formation com a àcid en dissolució)
  • Hidrurs no-metàl·lics especials: NH₃, PH₃, CH₄ (amb noms especials)

⚗️ Tres tipus de hidrurs

1️⃣ Hidrurs Metàl·lics

Metal + Hidrogen → Combinacions binàries binàries binàries binàries de metalls amb H

  • Nomenclatura: "Hidrur de [element]" o "Sèrie metàl·lica"
  • Exemples: LiH (hidrur de liti), CaH₂ (hidrur de calci)
  • Propietats: Sòlids iònics, reaccionen amb aigua

2️⃣ Hidràcids (No-metalls amb comportament àcid)

No-metall + Hidrogen → Solubles en aigua, formen àcids

  • Nomenclatura: "Àcid [no-metall]hídric"
  • Exemples: HCl (àcid clorhídric), H₂S (àcid sulfhídric)
  • Propietats: Gasos a temperatura ambient, solubles en H₂O

3️⃣ Hidrurs No-metàl·lics Especials

No-metalls especials + Hidrogen → Noms trivials comuns

  • NH₃ = Amoníac (o trihidrur de nitrogen)
  • PH₃ = Fosfina (o trihidrur de fòsfor)
  • CH₄ = Metà (o tetrahidrur de carboni)
  • H₂O = Aigua (o òxid de dihidrogen)

🧂 Sals Binàries (Combinacions Metal + No-metall)

📖 Què és una sal binària?

Una sal binària és un compost químic format per metall + no-metall. S'obtén generalment de la reacció entre un àcid i una base.

M + X → MXₙ
M = metall · X = no-metall · n = nombre d'àtoms del no-metall (segundo la valència)

La nomenclatura depèn de la valencia del metall i el no-metall.

⚗️ Com es formen les sals binàries?

Les sals es formen de diverses maneres:

Àcid + Base → Sal + Aigua
Exemple: HCl + NaOH → NaCl + H₂O
  • Reacció d'àcid-base: L'ió negatiu de l'àcid + ió positiu de la base
  • Metal + No-metall: Reacció directa entre els elements
  • Òxid metàl·lic + Àcid: MgO + 2HCl → MgCl₂ + H₂O

📝 Nomenclatura de Sals Binàries

El nom d'una sal binària està format per:

[Nom del no-metall + -ur/-id] + de + [Nom del metall + (valencia en romans si necessari)]

Regles de nomenclatura:

  • Fluorurs, Clorurs, Bromurs, Iodurs: El no-metall acaba en "-ur"
    • NaCl → Clorur de sodi
    • CaF₂ → Fluorur de calci (NO "difluorur" en nomenclatura Stock)
  • Sulfurs, Seleniurs, Telluurs: El no-metall acaba en "-ur"
    • Na₂S → Sulfur de sodi
    • Fe₂S₃ → Sulfur de ferro (III)
  • Nitrurs, Fosfurs, Carburs: El no-metall acaba en "-ur"
    • Mg₃N₂ → Nitrur de magnesi
    • AlP → Fosfur d'alumini

Diferència entre nomenclatures:

  • Stock: Nom + (valencia en romans si té més d'una) → "Sulfur de ferro (II)" o "Sulfur de ferro (III)"
  • Sistemàtica (IUPAC): Prefix + nom + -ur → "Disulfur de ferro", "Trisulfur de diferro"
