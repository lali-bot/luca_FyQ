⚛️ Valencias dels Elements

📖 Qué es la VALENCIA?

La valencia es el número de electrones que un elemento puede perdre, guanyar o compartir per formar un compost químic.

n = mínim de liens covalents o càrrega iònica
La valencia determina cómo los elementos es combinen!
  • Metalls: Pierden electrones → valencias POSITIVES (+1, +2, +3...)
  • No-metalls: Ganan electrones → valencias NEGATIVES (-1, -2, -3...)
  • Transitorials: Pueden tener MÚLTIPLES valencias

⚡ Cómo es determina la VALÈNCIA?

La posició a la Tabla Periódica determina la valencia:

  • Grup 1 (Alcalins): +1 (1 electron a la última capa)
  • Grup 2 (Alcalinoterris): +2 (2 electrons a la última capa)
  • Grup 13 (Boro, Alumini): +3 (3 electrons a la última capa)
  • Grup 14 (Carboni, Silici): +4 o -4 (4 electrons a la última capa)
  • Grup 15 (Nitrogen, Fòsfor): -3 o +3, +5 (5 electrons)
  • Grup 16 (Oxigen, Sofre): -2 o +4, +6 (6 electrons)
  • Grup 17 (Halògens): -1 o +1, +3, +5, +7 (7 electrons)
  • Grup 18 (Gasos nobles): 0 (no es combinen normalment)

🎯 Cómo USAR las valencias per fer fórmules

Cuando combinamos dos elements, les seves valencias han de "equilibrarse":

REGLA: El producte de la valencia × número de átomos tiene que ser igual per als dos elements.

Exemples:

  • Na (+1) + Cl (-1): 1×1 = 1×1 → NaCl ✓
  • Ca (+2) + Cl (-1): 1×2 = 2×1 → CaCl₂ ✓
  • Al (+3) + Cl (-1): 1×3 = 3×1 → AlCl₃ ✓
  • Fe (+3) + O (-2): 2×3 = 3×2 → Fe₂O₃ ✓
  • Ca (+2) + (OH)⁻ (-1): 1×2 = 2×1 → Ca(OH)₂ ✓

💡 Trucos PARA RECORDAR VALENCIES

  • GRUP = VALENCIA: Grup 1→ +1, Grup 2→ +2, Grup 13→ +3, etc.
  • Oxigen SEMPRE -2: (excepte en H₂O₂ que es -1)
  • Fluor SEMPRE -1: No canvia mai!
  • Metalls = positius: Els metalls siempre perden electrons
  • No-metalls = negatius: Els no-metalls siempre guanyen electrons
  • H con no-metalls = +1: (HCl, H₂S, NH₃)
  • H con metalls = -1: (LiH, MgH₂, AlH₃)
  • Transició = vàries: Fe(II) i Fe(III), Cu(I) i Cu(II)...
  • Tabla Periódica et diu tot: Mira la columna = valencia!

📚 Referencia de Hidruroos

🔥 Óxidos (Compostos con Oxigen)

📖 Qué es un óxido?

Un óxido es un compost químic format per un elemento + oxigen. Els óxidos son molt comuns a la natura.

Elemento + O₂ → Óxido (E₂Oₙ o EOₘ)
Depèn de les valencias de l'elemento i l'oxigen (-2)

Hi ha dos tipus principals:

  • Óxidos Metálicos (Óxidos Básicos): Metall + Oxigen → sòlids que formen bases
  • Óxidos No-metálicos (Anhídrids): No-metall + Oxigen → gasos o sòlids que formen àcids

⚗️ Cómo se forman los óxidos?

Els óxidos es formen de diverses maneres:

Elemento + O₂ → Óxido
Combustió o combinació directa
  • Combustió simple: 2Na + O₂ → Na₂O (sodi es crema)
  • Combustió incompleta: 2C + O₂ → 2CO (monóxido de carbono)
  • Combustió completa: C + O₂ → CO₂ (dióxido de carbono)
  • Reacció d'óxidos metálicos + aigua: Na₂O + H₂O → 2NaOH (forma base)
  • Reacció d'óxidos no-metálicos + aigua: SO₃ + H₂O → H₂SO₄ (forma àcid)

📝 Nomenclatura dels Óxidos

La nomenclatura dlos óxidos depèn del tipus:

🔹 Óxidos Metálicos (Bàsics)

Sistemática: (prefix) + óxido + de + nom element

  • Li₂O → Monóxido de dilíti (Sist) / Óxido de litio (Stock)
  • CaO → Monóxido de calcio (Sist) / Óxido de calcio (Stock)
  • Al₂O₃ → Trióxido de dialuminio (Sist) / Óxido de aluminio (Stock)
  • Fe₂O₃ → Trióxido de dihierro (Sist) / Óxido de hierro (III) (Stock)

🔹 Óxidos No-metálicos (Anhídrids)

Ambdós noms: Sistemática = Stock

  • CO₂ → Dióxido de carbono
  • SiO₂ → Dióxido de silicio
  • SO₂ → Dióxido de azufre
  • N₂O₅ → Pentóxido de dinitrógeno
  • Cl₂O₇ → Heptóxido de dicloro (màxim clor)

🧫 Hidroxids (Bases)

📖 Qué es un hidróxido?

Un hidróxido (o base) es un compost format per un metall + el grup hidroxil (OH⁻).

M(OH)n
M = metall · n = valencia del metal · OH⁻ = grup hidroxil

El grup OH siempre actua con càrrega −1. Per tant, el subíndex del grup OH coincideix con la valencia del metal.

⚗️ Cómo se forman?

Els hidróxidos es formen quan un óxido metálico reacciona con aigua:

Óxido metálico + H₂O → Hidróxido
Exemple: Na₂O + H₂O → 2 NaOH
  • El sodi (Na) tiene valencia +1 → NaOH (1 sol OH)
  • El calci (Ca) tiene valencia +2 → Ca(OH)₂ (2 grups OH)
  • El alumini (Al) tiene valencia +3 → Al(OH)₃ (3 grups OH)

📝 Nomenclatura

Hi ha dues maneres d'anomenar los hidróxidos:

1. Sistemática (IUPAC): Prefix numèric + "hidróxido de" + metall

  • NaOH → Hidróxido de sodio (mono- s'omet)
  • Ca(OH)₂ → Dihidróxido de calcio
  • Fe(OH)₃ → Trihidróxido de hierro

2. Stock: "Hidróxido de" + metall + (valencia en romans)

  • NaOH → Hidróxido de sodio (sense romans si solo tiene 1 valencia)
  • Fe(OH)₂ → Hidróxido de hierro (II)
  • Fe(OH)₃ → Hidróxido de hierro (III)

⚗️ Oxoácidos (Àcids Oxigenats)

📖 Qué es un oxoácido?

Un oxoácido (o àcid oxigenat) es un compost format per hidrogen (H) + no-metall + oxigen (O).

HxXOy
H = hidrogen · X = no-metall (N, S, Cl, P...) · O = oxigen

La nomenclatura dlos oxoácidos depèn de la valencia del no-metal. Utilitzem sufixos (-ós, -ic) i prefixos (hipo-, per-).

⚗️ Cómo se forman?

Els oxoácidos es formen quan un óxido no-metálico reacciona con aigua:

Óxido no-metálico + H₂O → Oxoácido
Exemple: Cl₂O₇ + H₂O → 2 HClO₄ (ácido perclórico)
  • CO₂ + H₂O → H₂CO₃ (ácido carbónico) — Present a les begudes gasoses!
  • SO₃ + H₂O → H₂SO₄ (ácido sulfúrico) — El más important industrial!
  • N₂O₅ + H₂O → HNO₃ (ácido nítrico) — Ús en síntesi química.
  • P₂O₅ + H₂O → H₃PO₄ (ácido fosfórico) — Ingredient en refrescos!

📝 Nomenclatura dels Oxoácidos

La nomenclatura depèn de la valencia del no-metal central. Per als no-metalls con multiple valencies, utilitzem prefixos i sufixos:

  • Hipo- + nom + -ós: Valencia más baixa (exemplo: HClO, hipoclorós)
  • Nom + -ós: Valencia intermèdia (exemple: HClO₂, clorós)
  • Nom + -ic: Valencia más alta (exemple: HClO₃, clòric)
  • Per- + nom + -ic: Valencia màxima (exemple: HClO₄, perclòric)

Observació: Per a no-metalls que solo tenemos 2 formes (com S), no fem servir hipo- ni per-.

🧪 Hidruros (Combinacions con Hidrogen)

📖 Qué es un hidruroo?

Un hidruro es un compost químic format per hidrogen (H) combinat con otro elemento (metal o no-metal).

XHn
X = qualsevol elemento · H = hidrogen · n = nombre d'hidrógens

Els hidruros es classifiquen en tres tipus segons el caràcter de l'elemento X:

  • Hidruros metálicos: Metal + H → LiH, NaH, CaH₂
  • Hidrácidos: No-metall + H → HCl, HBr, H₂S (formation cómo a àcid en dissolució)
  • Hidruros no-metálicos especials: NH₃, PH₃, CH₄ (amb noms especials)

⚗️ Tres tipus de hidruros

1️⃣ Hidruroos Metálicos

Metal + Hidrogen → Combinacions binàries binàries binàries binàries de metals con H

  • Nomenclatura: "Hidruro de [element]" o "Sèrie metálicoa"
  • Exemples: LiH (hidruro de litio), CaH₂ (hidruro de calcio)
  • Propietats: Sòlids iònics, reaccionen con aigua

2️⃣ Hidrácidos (No-metalls con comportament àcid)

No-metall + Hidrogen → Solubles en aigua, formen àcids

  • Nomenclatura: "Àcid [no-metall]hídric"
  • Exemples: HCl (ácido clorhídrico), H₂S (ácido sulfhídrico)
  • Propietats: Gasos a temperatura ambient, solubles en H₂O

3️⃣ Hidruros No-metálicos Especials

No-metalls especials + Hidrogen → Noms trivials comuns

  • NH₃ = Amoníaco (o trihidruroo de nitrógeno)
  • PH₃ = Fosfina (o trihidruroo de fósforo)
  • CH₄ = Metano (o tetrahidruroo de carbono)
  • H₂O = Agua (o óxido de dihidrógeno)

💡 Trucos para recordar

  • Metanol·lics: "Hidruro de [metall]" — format: MHₙ (n = valencia del metal)
  • Hidrácidos: "Àcid [no-metall]hídric" — gas que actua cómo a àcid en H₂O

🧂 Sals Binàries (Combinacions Metal + No-metall)

📖 Qué es una sal binaria?

Una sal binària es un compost químic format per metall + no-metall. S'obtienen generalment de la reacció entre un àcid i una base.

M + X → MXₙ
M = metall · X = no-metall · n = número de átomos del no-metal (segundo la valencia)

La nomenclatura depèn de la valencia del metal i el no-metall.

⚗️ Cómo se forman las sales binarias?

Les sals es formen de diverses maneres:

Àcid + Base → Sal + Agua
Exemple: HCl + NaOH → NaCl + H₂O
  • Reacció d'àcid-base: L'ió negatiu de l'àcid + ió positiu de la base
  • Metal + No-metall: Reacció directa entre los elementos
  • Óxido metálico + Àcid: MgO + 2HCl → MgCl₂ + H₂O

📝 Nomenclatura de Sals Binàries

El nom de una sal binària està format per:

[Nom del no-metal + -ur/-id] + de + [Nom del metal + (valencia en romans si necessari)]

Regles de nomenclatura:

  • Fluoruros, Cloruros, Bromuros, Yoduros: El no-metall acaba en "-ur"
    • NaCl → Cloruro de sodio
    • CaF₂ → Fluoruro de calcio (NO "difluoruro" en nomenclatura Stock)
  • Sulfuros, Seleniurs, Telluurs: El no-metall acaba en "-ur"
    • Na₂S → Sulfuro de sodio
    • Fe₂S₃ → Sulfuro de hierro (III)
  • Nitruros, Fosfuros, Carburos: El no-metall acaba en "-ur"
    • Mg₃N₂ → Nitruro de magnesio
    • AlP → Fosfuro de aluminio

Diferència entre nomenclatures:

  • Stock: Nom + (valencia en romans si tiene más de una) → "Sulfuro de hierro (II)" o "Sulfuro de hierro (III)"
  • Sistemática (IUPAC): Prefix + nom + -ur → "Disulfuroo de hierro", "Trisulfuroo de dihierro"

💡 Trucos para recordar

  • Fórmula: M actua con la seva + valea i X con la seva -valencia → intercanviar i simplificar!
  • Nomenclatura Stock: "[No-metall]-ur de [metall]" (amb romans si necessari)
  • Metalls con 1 sola valencia: No posem romans (Na, K, Ca, Mg, Al...)
  • Metalls con múltiples valencies: Siempre posem romans a Stock (Fe II/III, Cu I/II, Sn II/IV...)
  • Acolorits característics: FeS (negre), CuS (negre), PbS (negre), Fe₂S₃ (vermell)
